# Agent 智能体

## 核心构成

> **Agent = LLM + 规划 + 记忆 + 工具使用**

## 主要功能

### 规划（Planning）

- 将 **复杂任务拆解** 为可执行的子步骤。

### 记忆（Memory）

- **短期**：对话上下文、当前任务状态。
- **长期**：外部知识、用户偏好等（可结合向量库、数据库等）。

### 工具使用（Tool Use）

- 调用 **API / 插件** 连接现实世界。
- 典型能力：联网 **搜索**、**代码执行**、调用业务系统等。

## 应用场景

- 具有 **严格逻辑流** 的 **多步任务**。
- **业务流程自动化**（审批、运维、客服工单、数据分析流水线等）。
