## 二、AI生成视频的核心流程与原理

### 2.1 工作流总体结构

工作流以一个**主题提示词**作为起点，经过以下几个关键步骤：

1. **文本生成**  
   通过简短的提示词，调用RE模型生成与主题相关的故事文案（小短文）。

2. **文本拆分**  
   利用文本处理节点，以中文句号为分割符，将文案拆分成多条句子，形成句子数组。

3. **循环处理句子数组**  
   对每条句子循环执行以下操作：
   - 生成对应的图片提示词，并调用图像生成模型生成图片。
   - 生成对应的音频，通过音频合成插件生成朗读音频。
   - 获取每个音频的时长（derivation），用于后续视频时间轴的同步。

4. **素材格式化**  
   将生成的图片链接、音频链接、音频时长、字幕文本等多媒体信息，按照视频剪辑插件的要求格式化成JSON结构。

5. **视频草稿创建与编辑**  
   通过简映小助手插件，在服务器端创建视频草稿（Draft），并依次添加图片、音频、字幕等素材。

6. **视频生成**  
   草稿保存后，调用生成视频接口，输出最终视频文件链接。

### 2.2 关键技术点与插件说明

- **简映小助手插件**  
  该插件是整个视频生成的核心，提供了创建草稿、添加图片、添加音频、添加字幕、保存草稿、生成视频等多种功能。其原理是在服务器端调用剪映视频编辑工具，将素材自动导入并合成视频。

- **素材时间轴同步**  
  每张图片的显示时长略长于对应音频的播放时长，图片和音频的起止时间精确对应，保证视频画面与声音同步。音频之间通常会留有间隔（如1秒），以保证节奏感。

- **多媒体格式化**  
  通过Python代码将素材信息转换为插件要求的复杂JSON格式，包含每张图片的URL、尺寸、时间区间，音频的URL、起止时间，字幕文本及其样式（字体、颜色、位置等）。

## 三、详细流程解析

### 3.1 文本生成与拆分

- 输入示例主题：  
  *“AI产品经理借助AI的能力快速成长，帮助企业迎接AI的机会”*

- 生成结果为一段包含约10句的中文短文，每句以中文句号结束。

- 通过文本处理节点，以句号为分割符，将文案拆分为10条句子，形成数组。

### 3.2 图片与音频生成

- 对每条句子生成对应的**英文提示词**，用于调用图像生成模型生成图片。例如：

  | 中文句子                         | 英文提示词示例                                 |
  |---------------------------------|----------------------------------------------|
  | AI工具可快速分析用户需求         | Data-driven decision making shortens dev cycle |

- 生成10张图片，分别对应10句文案。

- 使用火山引擎音频合成插件，将每句文本合成为音频，获取音频链接及时长（derivation）。

- 生成的图片和音频数量应一致，若有生成失败（如图片安全审核未通过），则数量会减少。

### 3.3 多媒体素材格式化示例（JSON结构）

```json
{
  "Captions": [
    {"text": "AI产品经理借助AI的能力快速成长。"},
    ...
  ],
  "Images": [
    {
      "url": "https://example.com/image1.jpg",
      "width": 920,
      "height": 1536,
      "start": 0,
      "end": 4.48
    },
    ...
  ],
  "Audios": [
    {
      "url": "https://example.com/audio1.mp3",
      "derivation": 3.48,
      "start": 0,
      "end": 3.48
    },
    ...
  ]
}
```

- 图片的显示时间段通常覆盖对应音频播放时间，并略有延长。
- 音频播放时间之间留有间隔，保证视频节奏自然。

### 3.4 草稿创建与视频生成

- **创建草稿**：调用插件的`create draft`接口，指定视频宽高（如920x1536），获得草稿URL。

- **添加素材**：依次调用`add image`、`add audio`、`add caption`接口，将格式化后的图片、音频、字幕添加到草稿中。

- **保存草稿**：调用保存接口，草稿以JSON文件形式存储，兼容剪映视频编辑器。

- **生成视频**：调用`gen video`接口，生成最终视频文件，返回视频链接。

## 六、注意事项与实践建议

- **API Key管理**  
  生成完整视频时需提供简映小助手插件的API Key，部分Key有调用次数或余额限制，建议注册个人账号并充值以保证稳定运行。

- **素材安全审核**  
  图片生成可能因内容安全审核失败导致素材缺失，需做好异常处理。

- **格式规范**  
  多媒体素材必须严格按照插件文档格式组织，建议使用Python脚本自动转换。

- **草稿与视频关系**  
  工作流生成的是视频草稿（JSON格式），可导入剪映继续编辑，非直接生成最终视频文件，提升创作灵活性。

- **动效与背景音乐**  
  复杂视频需调用额外API（如海螺视频）实现动效，背景音乐需单独添加，提升视频表现力。
