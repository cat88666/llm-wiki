# 正则化

> 一句话定义：正则化通过在损失函数中加入惩罚项限制模型复杂度，防止过拟合；L1 产生稀疏解适合特征选择，L2 权重均匀缩小适合防过拟合。

## 核心要点
- L1 正则（Lasso）：惩罚项为 λ·Σ|w|；梯度是常数（±λ），驱使小权重直接归零，产生稀疏解；适合高维特征选择，自动丢弃无关特征
- L2 正则（Ridge）：惩罚项为 λ·Σw²；梯度正比于权重（-2λw），权重越大惩罚越强，最终权重均匀缩小但不为零；适合防止过拟合同时保留所有特征
- Elastic Net：L1 + L2 的线性组合（α·L1 + (1-α)·L2），兼顾稀疏性和稳定性；当特征高度相关时优于纯 L1（纯 L1 在相关特征中随机选一个，Elastic Net 倾向于同时保留）
- XGBoost 的正则化：在目标函数中直接加入 γ·T（叶节点数惩罚）+ λ·Σw²（L2）+ α·Σ|w|（L1），其中 γ 控制树的复杂度，比传统后剪枝更系统
- 神经网络中的 Dropout 是另一种正则化：训练时随机丢弃神经元（等效于训练多个子网络的集成），推理时使用全部神经元并缩放权重

## 详细说明

L1 和 L2 的本质差异来源于惩罚项的几何形状：L1 的约束域是菱形（顶点在坐标轴上），最优解容易落在顶点（某些权重恰好为零）；L2 的约束域是球形（光滑），最优解通常在非零点上与等高线相切。这个几何直觉解释了为什么 L1 产生稀疏解而 L2 不产生稀疏解。

正则化强度 λ 是关键超参数：λ 过小，正则化效果弱，模型过拟合；λ 过大，模型欠拟合（所有权重被压缩到接近零）。通常通过交叉验证选择最优 λ，搜索范围在对数空间（如 0.001, 0.01, 0.1, 1, 10）。

在深度学习中，L2 正则化等价于权重衰减（Weight Decay）：每次参数更新时额外乘以 (1 - ηλ)，直接在优化器中实现。AdamW 是 Adam + 解耦 Weight Decay（避免 Adam 的自适应学习率影响正则化效果），是目前 LLM 训练的主流优化器。

## 与其他概念的关系
- 区别于 [[梯度下降与优化]]：正则化修改损失函数（约束参数空间），优化方法决定如何更新参数，两者正交
- 延伸自 [[监督学习与无监督学习]]：正则化是监督学习中控制偏差-方差权衡的核心手段
- 关联 [[集成学习]]：XGBoost 将 L1/L2 正则直接嵌入树的增益公式，是正则化在集成方法中的深度应用

## 应用场景
- 高维稀疏特征（文本 TF-IDF、用户行为 one-hot）：L1 自动选择有效特征，减少模型复杂度
- 特征高度相关（多重共线性）：L2 稳定地缩小所有相关特征权重，Elastic Net 效果更好
- LLM 微调：LoRA 的低秩约束是一种隐式正则化（限制参数更新的秩），防止小数据集微调过拟合
- 神经网络训练：Dropout（0.1-0.5）+ Weight Decay（1e-4 到 1e-2）是标准组合
