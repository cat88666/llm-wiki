# Agent

## 定义
Agent 是具备规划、调用工具、观察反馈和持续执行能力的模型系统。

## 解决的问题
- 多步任务拆解
- 工具链协同
- 任务状态持续推进

## 核心机制
- 目标理解
- 计划生成
- 工具调用
- 观察与继续执行

## 关键判断
- 单步问题不该被滥用成 Agent
