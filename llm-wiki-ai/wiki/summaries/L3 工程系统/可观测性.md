# 可观测性

## 定义
可观测性是对 LLM 系统的输入、检索、模型、工具、成本和错误进行链路追踪的能力。

## 解决的问题
- 无法定位问题出在哪一层
- 无法解释效果波动
- 无法持续跟踪成本与延迟

## 核心机制
- Trace ID
- 模型与 Prompt 版本记录
- Token 与时延指标
- 工具与检索日志

## 关键判断
- 没有可观测性，生产化优化基本无法闭环
