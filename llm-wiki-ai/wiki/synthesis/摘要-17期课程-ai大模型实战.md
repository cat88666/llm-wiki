## 核心摘要

这份资料本质上是一张「大模型应用开发训练营学习地图」，把 Prompt、RAG、Agent、Transformer、微调、多模态、部署交付等主题串成一条完整学习路径，并为每个课次给出可替代的免费公开资源。

**核心价值：**
- 把大模型应用学习拆成基础入门、核心技术、进阶实战、综合项目四个阶段
- 覆盖 Prompt、RAG、Agent、Transformer、微调、多模态、部署等完整技术栈
- 提供 Datawhale、LangChain、Hugging Face、LlamaIndex、LLaMA-Factory 等公开资源入口
- 强调“做项目比听课更重要”，适合作为自学路线图和资料总导航

**建议学习主线：**
1. 先打基础：Python、Prompt、基础 API 调用
2. 再补核心：RAG、Agent、Transformer
3. 然后实战：微调、部署、工程交付
4. 最后产出：做完整项目、写简历、积累 GitHub 记录
