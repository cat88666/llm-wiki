## 核心摘要

这份课程笔记把 Agent 定义为 `LLM + 规划 + 记忆 + 工具使用` 的组合体，强调 Agent 不是单次回答优化，而是面向复杂多步任务的系统能力。

**关键要点：**
- 规划负责把复杂任务拆成子步骤
- 记忆分为短期状态和长期外部知识
- 工具使用让模型连接搜索、代码执行和业务系统
- Agent 适合严格逻辑流、多步协同、流程自动化等场景
